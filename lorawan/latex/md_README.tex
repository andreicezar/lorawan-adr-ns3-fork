\hypertarget{md_README_autotoc_md2}{}\doxysubsection{Prerequisites}\label{md_README_autotoc_md2}
To run simulations using this module, you first need to install ns-\/3. If you are on Ubuntu/\+Debian/\+Mint, you can install the minimal required packages as follows\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{sudo apt install g++ python3 cmake ninja-\/build git ccache}

\end{DoxyCode}


Otherwise please directly refer to the \href{https://www.nsnam.org/wiki/Installation\#Prerequisites}{\texttt{ prerequisites section of the ns-\/3 installation page}}.

\begin{quote}
Note\+: While the {\ttfamily ccache} package is not strictly required, it is highly recommended. It can significantly enhance future compilation times by saving tens of minutes, albeit with a higher disk space cost of approximately 5GB. This disk space usage can be eventually reduced through a setting. \end{quote}
Then, you need to\+:


\begin{DoxyEnumerate}
\item Clone the main ns-\/3 codebase,
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Clone this repository inside the {\ttfamily src} directory therein, and
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Checkout the current ns-\/3 version supported by this module.
\end{DoxyEnumerate}

To install this module at the latest commit, you can use the following all-\/in-\/one command\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{git clone https://gitlab.com/nsnam/ns-\/3-\/dev.git \&\& cd ns-\/3-\/dev \&\&}
\DoxyCodeLine{git clone https://github.com/signetlabdei/lorawan src/lorawan \&\&}
\DoxyCodeLine{tag=\$(< src/lorawan/NS3-\/VERSION) \&\& tag=\$\{tag\#release \} \&\& git checkout \$tag -\/b \$tag}

\end{DoxyCode}


{\bfseries{Note}}\+: When switching to any previous commit, {\itshape including the latest release}, always make sure to also checkout ns-\/3 to the correct version ({\ttfamily NS3-\/\+VERSION} file at the root of this repository) supported at that point in time.\hypertarget{md_README_autotoc_md3}{}\doxysubsection{Compilation}\label{md_README_autotoc_md3}
Ns-\/3 adopts a development-\/oriented philosophy. Before you can run anything, you\textquotesingle{}ll need to compile the ns-\/3 code. You have two options\+:


\begin{DoxyEnumerate}
\item {\bfseries{Compile ns-\/3 as a whole\+:}} Make all simulation modules available by configuring and building as follows (ensure you are in the {\ttfamily ns-\/3-\/dev} folder!)\+:
\end{DoxyEnumerate}


\begin{DoxyCode}{0}
\DoxyCodeLine{./ns3 configure -\/-\/enable-\/tests -\/-\/enable-\/examples \&\&}
\DoxyCodeLine{./ns3 build}

\end{DoxyCode}



\begin{DoxyEnumerate}
\item {\bfseries{Focus exclusively on the lorawan module\+:}} To expedite the compilation process, as it can take more than 30/40 minutes on slow hardware, change the configuration as follows\+:
\end{DoxyEnumerate}


\begin{DoxyCode}{0}
\DoxyCodeLine{./ns3 clean \&\&}
\DoxyCodeLine{./ns3 configure -\/-\/enable-\/tests -\/-\/enable-\/examples -\/-\/enable-\/modules lorawan \&\&}
\DoxyCodeLine{./ns3 build}

\end{DoxyCode}


The first line ensures you start from a clean build state.

Finally, ensure tests run smoothly with\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{./test.py}

\end{DoxyCode}


If the script reports that all tests passed you are good to go.

If some tests fail or crash, consider filing an issue.\hypertarget{md_README_autotoc_md4}{}\doxysection{Usage examples}\label{md_README_autotoc_md4}
